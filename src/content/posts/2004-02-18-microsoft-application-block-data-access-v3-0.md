Dat wist ik dus persoonlijk niet, maar het blijkt dus dat versie 3.0 van de Data Access Application Block alweer sinds 11 augustus 2003 uit is. Ik loop dus nogal achter.

Deze versie ondersteunt naast de SQLServer ADO.NET driver, ook OleDB en ODBC versies, waardoor je ook andere databases kunt benaderen zoals Oracle, Access, etc.

Op MSDN kun je er nog niets over vinden, hij’s echter al wel op [gotdotnet te downloaden](http://www.gotdotnet.com/Community/Workspaces/workspace.aspx?id=c20d12b0-af52-402b-9b7c-aaeb21d1f431).

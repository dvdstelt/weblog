On march 26, 2007 it was announced that [TeamPlain](http://www.devbiz.com/teamplain/webaccess/default.aspx) is now freely downloadable for Team Foundation Server customers as a result of Microsoft’s acquisition of [devBiz](http://www.devbiz.com/), creators of this excellent product. This is great news, as customers who paid a lot of money for TFS had to lay down some extra money (per user) for TeamPlain licenses.

TeamPlain enables webbased access to Team Foundation Server features; work-items, documents and reports. Remember that every user still needs a Client Access License (CAL). But if you’re already using TFS, be sure to [download the now free TeamPlain](http://www.devbiz.com/teamplain/webaccess/download.aspx).

While Windows Workflow Foundation (WF) already had [its own website](http://www.windowsworkflow.net/) for a while, Windows Communication Foundation (WCF) now also has [its own website](http://windowscommunication.net/). It has some samples, some extensions, links to the MSDN forum and provides resources to some alternative websites with webcasts and more alike.

Awesome! Just a few days before the .NET 3.5 Summer Class, Silverlight 1.0 is released.

With [the announcement](http://weblogs.asp.net/scottgu/archive/2007/09/04/silverlight-1-0-released-and-silverlight-for-linux-announced.aspx), the news is out that Silverlight will be officially supported by Microsoft and Novell on Linux via the Mono initiative. Microsoft will deliver Silverlight Media Codecs for Linux and Novell will be building a 100% compatible Silverlight runtime implementation called “Moonlight”.

You can read much more about Moonlight on [Miguel de Icaza’s weblog](http://tirania.org/blog/archive/2007/Sep-05.html) and [view some screenshots here](http://www.mono-project.com/MoonlightShots).

I’m pretty excited about Silverlight. Before Silverlight I was never really interested in WPF’s XAML, but I’m learning fast now and I’m loving it more and more.

![Win95 on pocketpc](/images/windows-95-98-makes-it-to-pocket-pc/bochs9.jpg)This is definitly something I would try to install on my PocketPC, if I had one! Some guys seem to have tried to get Win98 and Win95 running on PocketPCs.

If you want to install this, be sure to have a 256MB minimum on memory, but as it seems you might also be able to stream the image over a wireless network. Then you need [<font color="#0000cc">Nyditot Virtual Display</font>](http://www.nyditot.com/) to increase your screen resolution. (This might be interesting to more people then those keen on installing Win98.) You’ll also need [an emulator](http://mamaich.kasone.com/) to run DOS.

More information [can be found here](http://www.pocketgamer.org/showthread.php?s=&threadid=3660).

Have fun installing! 😉

Ik wil een FTP server opzetten zodat de bloggers ook files kunnen uploaden, zodra ze dat nodig vinden. Ik heb alleen geen tijd om alle FTP servers even te testen.

Heeft iemand ervaringen met een FTP server? Zo ja, zet je bevindingen of favoriete applicatie in de reacties.  
Mijn versie moet aan de volgende eisen voldoen:
* Windows 2000 compatible  
* Per user een directory aan kunnen geven  
* Niet te ingewikkeld 😉

Ik zie op Jan Tielens zijn weblog iets wat wellicht interessant is voor onze SharePoint mannen (en vrouwen).

In SharePoint blijk je alles, buiten de web interface, ook via web services te kunnen doen. En dat is iets wat ik persoonlijk het leuke vind van web services, je kunt een Windows Forms client bouwen en die met de webservices laten praten. Da’s allemaal driedubbel werk (web interface, web services én je winforms client) maar dan heb je ook wel iets waanzinnig moois wat betreft functionaliteit!

Jan heeft een voorbeeld applicatie gebouwd waarin hij contact legt met SharePoint d.m.v. deze webservices.

* NerdDinner for Visual Studio 2008 in folder structure as specified in presentation 
* FinalBuilder 6 script for building NerdDinner 
* FinalBuilder 6 script for deploying a ClickOnce application as described in [this article](https://bloggingabout.net/2008/11/26/deploying-clickonce-applications-automated-using-finalbuilder).
